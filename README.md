# 看懂卢浮宫｜Louvre AI Guide

面向中国青年游客的卢浮宫中文 AI 导览产品 Demo。

这是一款帮助用户在有限时间内真正看懂卢浮宫的移动端 Web App 原型，而不是只打卡《蒙娜丽莎》。产品以「简约艺术导览手册」为视觉方向，串联游前预习、主题路线、区域级地图、作品讲解、AI 识别模拟和游览总结等核心体验。

## 核心功能

- 游前预习
- 主题路线推荐
- 区域级分层地图
- 作品详情讲解
- AI 识别模拟
- 游览徽章和总结

## 技术栈

- Next.js
- TypeScript
- Tailwind CSS
- 本地 JSON mock 数据

## 如何运行

```bash
npm install
npm run dev
```

启动后在浏览器中打开本地开发地址即可体验 Demo。

## Demo 流程

启动页 → 博物馆选择 → 卢浮宫首页 → 路线推荐 → 地图 → 作品详情 → AI 识别 → 徽章总结

## MVP 说明

当前版本为作品集 Demo，不包含真实室内定位、真实 AI 图像识别和真实用户系统。

所有数据均来自本地 JSON mock 文件，AI 识别和播放讲解等交互为前端模拟，用于展示核心产品流程和体验设计。
